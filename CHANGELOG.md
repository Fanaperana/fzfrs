# Changelog

All notable changes to this project will be documented in this file.

## 0.1.0 - 2026-04-03

First public release of fuzzly.

- Added educational Rust implementations of Levenshtein distance and fuzzy search
- Added benchmark coverage with Criterion and documented performance results
- Added optimized implementations including reduced-memory and fast-path variants
- Added a polished documentation site published with GitHub Pages
- Added repository branding, SEO metadata, and social preview support
- Added community standards, discussion templates, issue templates, and PR template
