## 🧪 What is this?

**fzfrs** is an educational deep-dive into fuzzy search algorithms. It implements Levenshtein distance-based matching in Rust with clear code, pseudocode, and examples — so you can understand and port it to any language.

## ✨ Features

- 📐 Levenshtein distance implementation from scratch
- 🔍 Fuzzy string matching engine
- 📚 Pseudocode + comments for every algorithm step
- 🦀 Idiomatic Rust — good reference for learning

## 🚀 Usage

```bash
cargo run
cargo test
```

## 📖 Learn

Start with `src/lib.rs` for the core algorithm, then `src/main.rs` for example usage. Key concepts: edit distance, dynamic programming, fuzzy match scoring.
